# 抖音IP属地助手

在抖音网页版视频旁显示作者IP属地。

## 效果

视频发布时间右边会出现 🌍 标签，显示作者当前的 IP 省份。

## 安装方法

1. 下载 `douyin-ip-locator.zip` 并解压
2. 打开 Chrome，地址栏输入 `chrome://extensions/`（Edge 则是 `edge://extensions/`）
3. 打开右上角「开发者模式」
4. 点击「加载已解压的扩展」
5. 选择解压后的 `douyin-ip-locator` 文件夹
6. 打开 [抖音网页版](https://www.douyin.com) 即可看到效果

## 工作原理

- 从抖音视频右侧信息面板识别当前作者
- 访问作者主页 HTML 提取 IP 属地
- 以浮动标签形式显示在视频发布时间旁

## 说明

- 显示的 IP 为作者**当前**IP 属地（与作者主页显示一致）
- 仅支持 Chrome / Edge 浏览器
- 不收集任何用户数据，所有请求均为直接访问抖音服务器
